À propos

GéniALE a pour principale mission de développer des technologies dans le domaine brassicole afin d’automatiser et optimiser les processus de brassage.

En collaboration avec des grands joueurs de l’industrie, le club se concentre sur le développement de nouvelles technologies dans le but d'améliorer les procédés existants.

Banc de brassage

Le banc de brassage permette de brasser de faible quatité de bière pour des brassins tests ou des brassins d'exeception. De plus, c'est sur ce système qu'on développe nos différentes technologique avec le boitier électrique, le système de contrôle, l'automatisation et les interfaces de contrôle).

  • Client: Start Bootstrap
  • Date: April 2014
  • Service: Web Development

Boitier électrique

Le boitier électique acccueil une entrée de 220V. On y retrouve les relais, les disjoncteurs, le communutateur ainsi que le système de controle.

  • Client: Start Bootstrap
  • Date: April 2014
  • Service: Web Development

Systeme de controle

Le systeme de controle permette de controler et d'acquissition le systeme de brassage. Fonctionnant sur le principe de l'internet des objets, il est possible d'y accéder via une page web, un application mobile puisqu'il est connecter au réseau internet. Ainsi la configuration et le monitorage ont la possibilité d'être fait de partout. Le syteme est entierèment personnalisable en changeant les différentes cartes esclaves permmettant ainsi de s'adapter au besoin des différents systemes de brassage.

  • Client: Start Bootstrap
  • Date: April 2014
  • Service: Web Development

Présentoire

Le présentoire est un bar à bière portatif. Il permet de servir jusqu'à 2 bières refroidies en fût. Nous sommes partiellement fier de servir les bières que le club brassage avec notre présentoire. Des améliorations sont prévues pour l'années en cours.

  • Client: Start Bootstrap
  • Date: April 2014
  • Service: Web Development

Projet de développement durable

Le projet a pour but d’optimiser la consommation énergétique des procédés et de réduire au maximum la consommation d’eau potable. Dans cette perspective, l’équipe d’ABD a mis en place un système de récupération d’eau de pluie. Pour sa part, GéniALE a fait la conception d’un nouveau système de refroidissement, a évalué la possibilité de faire de la récupération de chaleur par les gaz de combustion de la chaudière et a aussi exploré la rentabilité d’installer un système de récupération du CO2 avec ceux présentement disponibles sur le marché.

  • Client: Start Bootstrap
  • Date: April 2014
  • Service: Web Development

Saveurs de génie

Saveurs de génie est une compétition interuniversitaire de clubs brassicole. Cette compétition comprend pour le développement technologique et un volet dégustatif pour la bière. En 2015, GéniAle a remporté la première place pour le volet technologique. En 2016, GeniAle a remporter la première place pour le volet dégustatif et a partager la première place pour le volet technologique.

  • Client: Start Bootstrap
  • Date: April 2014
  • Service: Web Development

La Page Blanche

La Page Blanche est une bière de blé traditionnelle bavaroise brassée à l'aide d'une souche de levure allemande, ce qui la distingue par ses arômes de banane et de clou de giroffe. Ces arômes s'harmonisent à merveille avec la texture pétillante de la bière afin de donner une bière fruitée et rafraîchissante. Santé!

  • Type: Weizen
  • Date: Novembre 2016
  • Évènement: Saveurs De Génie
